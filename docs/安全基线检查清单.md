## 三、生产环境建议

### 3.1 必须修改的配置

```bash
# 环境变量设置（生产环境）
export MYSQL_ROOT_PASSWORD=<强密码>
export MYSQL_PASSWORD=<强密码>
export RABBITMQ_DEFAULT_PASS=<强密码>
export NACOS_AUTH_ENABLE=true
export NACOS_AUTH_TOKEN=<随机密钥>
```

### 3.2 网络隔离

- 数据库、Redis、RabbitMQ 仅允许内网访问
- 仅暴露网关端口（8080）给外部
- 使用 HTTPS（需配置 SSL 证书）

### 3.3 敏感信息存储

- 使用密钥管理服务（如 Vault、阿里云 KMS）
- 或使用 Kubernetes Secrets
- 避免将密钥存储在代码或配置文件中
